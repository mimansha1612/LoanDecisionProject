# LoanDecisionProject
# Policy Optimization for Loan Approvals

## Steps to Run

1. Install requirements: pip install -r requirements.txt
2. Open notebooks/EDA_DL_RL.ipynb in Jupyter or Colab
3. Run cells in order:
   - Task 1: EDA & Preprocessing
   - Task 2: Deep Learning Model
   - Task 3: Offline RL Policy
4. Check final outputs:
   - DL model: AUC & F1
   - RL agent: Avg Reward
